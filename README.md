# Анализ предпринимателей ЕГРЮЛ: Дашборд в Apache Superset

## О проекте
Этот проект представляет собой комплексное исследование данных о российских предпринимателях из Единого государственного реестра юридических лиц (ЕГРЮЛ). **Цель проекта** - создать интерактивный дашборд, который наглядно ответит на основные вопросы (1. Насколько активно регистрируется новый бизнес в России? Какую форму собственности и вид деятельности выбирают? 2.  Как обстоят дела с ликвидацией бизнеса? 3. Сколько "живет" бизнес в России?)

**Стек технологий:**
- **Анализ:** Python (Pandas, NumPy) в Google Colab
- **Визуализация:** Apache Superset
- **Источник данных:** выборка в 500 тыс. предпринимателей из всего реестра ЕГРЮЛ/ЕГРИП

### 1. Разведочный анализ данных (EDA) в Google Colab
1. **Загрузка данных**
   - Импорт исходного CSV-файла
2. **Очистка данных**
   - Работа с пропусками (удаление/заполнение)
   - Приведение типов (даты, числовые поля)
   - Нормализация и переименование столбцов, где это необходимо
3. **Экспорт итогового датафрейма**
   - Сохранение обработанного датафрейма в CSV для дальнейшего импорта в Superset


### 2. Загрузка в Superset
После завершения обработки данных:

1. Итоговый CSV импортируется в Superset как источник данных
2. Настраиваются измерения и метрики
3. На основе источника создаётся интерактивный дашборд
